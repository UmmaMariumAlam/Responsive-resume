A fully responsive personal portfolio built with HTML, CSS, and JavaScript.

Live at: [ummamariumalam.github.io/Responsive-resume/](https://ummamariumalam.github.io/Responsive-resume/)

---

### 📄 LaTeX CV (`latex-cv/`)
A version of my resume written and compiled in LaTeX using
[Prism](https://prism.openai.com) (OpenAI's cloud-based LaTeX editor).
Supports both a dark-mode and a light/print-friendly mode via a single
theme switch in the source file.

→ See [`latex-cv/README.md`](latex-cv/README.md) for details on compiling
and switching themes.

---

### 🖥️ HTML CV Generator (`html-cv-generator/`)
A small tool that generates a pixel-accurate PDF of my CV directly from
an HTML/CSS layout — preserving the dark-mode design using Playwright,
instead of falling back to a plain white background like browser print exports.

→ See [`html-cv-generator/README.md`](html-cv-generator/README.md) for
setup and usage.
